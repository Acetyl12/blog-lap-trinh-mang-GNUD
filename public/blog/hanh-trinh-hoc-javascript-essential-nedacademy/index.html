Hành trình học JavaScript Essential 1 & 2 (NED Academy)

Posted on October 21, 2025  •  6 minutes  • 1143 words

Bài viết này là ghi chép “thực chiến” sau khi mình học xong JavaScript Essential 1 & 2 trên NED Academy. Mục tiêu là nắm chắc nền tảng JS và có đủ tự tin để làm các mini-project, sau đó đi sâu hơn vào async, module, DOM APIs và testing.

Tại sao chọn khóa Essential 1 & 2?

Nền tảng vững chắc: Nắm chắc biến, kiểu dữ liệu, hàm, mảng/đối tượng, scope, this, DOM, và bất đồng bộ. Bài giảng ngắn gọn: Mỗi bài 10–15 phút, ví dụ thực tế, phù hợp khi bận rộn. Học đều đặn: Mỗi ngày 45–60 phút, tránh “cram” kiến thức.

Lộ trình học & cách thực hiện

Essential 1: Biến, kiểu dữ liệu, toán tử, điều kiện, vòng lặp, hàm cơ bản.

Mục tiêu: Viết script xử lý dữ liệu, hiểu let/const, === vs ==.

Essential 2: Mảng/đối tượng, hàm nâng cao, scope/hoisting, this, DOM, async cơ bản.

Mục tiêu: Thao tác DOM, gọi API với fetch, viết mini app.

Phương pháp học:

Ghi note ngắn (1 trang/module). Viết 1 snippet minh họa ý chính. Làm mini-project (30–60 phút) để áp dụng.

Kiến thức cốt lõi – Essential 1

1) Biến & kiểu dữ liệu

  • Dùng const mặc định; chỉ let khi cần gán lại; tránh var.
  • Nhớ typeof null === "object" (di sản lịch sử JS 😅).
const PI = 3.14;
let count = 0; // sẽ thay đổi
// var old = 'avoid'; // hoisting + function scope gây lỗi khó debug
Truthy/Falsy quan trọng khi viết điều kiện:

js
Sao chép mã
// Falsy: false, 0, "", null, undefined, NaN
if ("0") console.log("Chuỗi '0' là truthy"); // sẽ chạy
2) Toán tử & so sánh
Ưu tiên === (so sánh nghiêm ngặt) thay vì ==.

js
Sao chép mã
console.log(0 == false);  // true (ép kiểu)
console.log(0 === false); // false (không ép kiểu)
3) Điều kiện & vòng lặp
js
Sao chép mã
const score = 78;
const level = score >= 90 ? "A" : score >= 75 ? "B" : "C";

for (let i = 0; i < 3; i++) console.log(i);
4) Hàm cơ bản
Khai báo hàm, tham số mặc định, trả về sớm.

js
Sao chép mã
function greet(name = "bạn") {
  return `Xin chào, ${name}!`;
}
console.log(greet("Dung"));
Lỗi mình từng gặp: quên return, hoặc gán giá trị vào biến const.

Kiến thức cốt lõi – Essential 2
1) Mảng & phương thức quan trọng
map (biến đổi), filter (lọc), reduce (gộp), find (tìm 1 phần tử), some/every.

js
Sao chép mã
const users = [
  { name: "A", age: 20 },
  { name: "B", age: 28 },
  { name: "C", age: 17 },
];
const adults = users.filter(u => u.age >= 18).map(u => u.name);
console.log(adults); // ["A", "B"]
2) Đối tượng, destructuring & spread
js
Sao chép mã
const person = { name: "Dung", city: "TP.HCM", role: "dev" };
const { name, ...rest } = person; // rest: {city, role}
const clone = { ...person, role: "network-dev" };
3) Hàm nâng cao, scope & hoisting
Arrow function: ngắn gọn, không có this riêng.

Hoisting: var và function declaration được “kéo lên đầu scope”.

js
Sao chép mã
say(); // OK, hoisting với function declaration
function say() { console.log("Hello"); }

// Arrow function thì KHÔNG hoisting khả dụng:
try {
  hi(); // ReferenceError
} catch (e) { console.log(e.message); }
const hi = () => console.log("Hi");
4) this và ngữ cảnh
Trong event handler/method, cẩn thận context.

Arrow function kế thừa this của scope ngoài.

js
Sao chép mã
const obj = {
  value: 42,
  run() {
    setTimeout(() => {
      console.log(this.value); // 42 (arrow kế thừa this của run)
    }, 0);
  }
};
obj.run();
5) DOM & Events (cơ bản)
js
Sao chép mã
const list = document.querySelector("#todo-list");
document.querySelector("#add").addEventListener("click", () => {
  const li = document.createElement("li");
  li.textContent = "New item";
  list.appendChild(li);
});

// Ủy quyền sự kiện (event delegation)
list.addEventListener("click", e => {
  if (e.target.matches("li")) e.target.remove();
});
6) Bất đồng bộ: fetch, Promise, async/await
js
Sao chép mã
async function loadUsers() {
  try {
    const res = await fetch("https://jsonplaceholder.typicode.com/users");
    if (!res.ok) throw new Error(res.status);
    const data = await res.json();
    console.log(data.map(u => u.name));
  } catch (err) {
    console.error("Fetch lỗi:", err.message);
  }
}
loadUsers();
Mini-projects mình đã làm
Todo List (localStorage)

Thêm/xóa/click hoàn thành; tự lưu trạng thái:

js
Sao chép mã
const KEY = "todos";
const todos = JSON.parse(localStorage.getItem(KEY) || "[]");
function save() { localStorage.setItem(KEY, JSON.stringify(todos)); }
Countdown Timer

setInterval, dừng khi đến 0, định dạng mm:ss.

Search Debounce

Chỉ gọi API sau khi người dùng dừng gõ 300ms:

js
Sao chép mã
const debounce = (fn, ms=300) => {
  let t; return (...args) => { clearTimeout(t); t = setTimeout(() => fn(...args), ms); };
};
Gallery fetch ảnh

Gọi API, render lưới ảnh, lazy-load.

Mỗi mini-project 30–90 phút đủ giúp “đóng kiến thức”.

Lỗi hay gặp & cách mình sửa
Quên ===: dẫn tới bug khó hiểu → rule: luôn === trừ khi có chủ đích.

Hoisting var: “biến nhảy múa” → tránh var, dùng let/const.

Sao chép mảng/đối tượng nông (shallow): dùng ...spread hoặc structuredClone khi cần.

Event không chạy vì selector sai → mở DevTools → tab Elements kiểm tra.

Promise chưa await: console log ra Promise {…} → thêm await hoặc .then.

Cách mình ôn tập & nhớ lâu
Spaced repetition: 10–15 flashcards (khái niệm & trick).

Write to learn: viết blog ngắn sau mỗi module.

Build to learn: biến bài tập cuối module thành mini-app (có UI thật).

Review code: nhìn lại “code cũ” sau 1 tuần → refactor với kiến thức mới.

Gợi ý tài nguyên kèm theo
MDN Web Docs (JS & DOM) – tra cứu chuẩn & ví dụ.

JavaScript.info – diễn giải dễ hiểu, nhiều bài tập.

Eloquent JavaScript – đọc thêm khi muốn hiểu sâu.

Kế hoạch tiếp theo
Tối ưu mini-projects (module hóa code, tách file).

Học thêm TypeScript (type an toàn), Jest (unit test).

Thử build tính năng realtime nhỏ với WebSocket (nối với mảng networking mình yêu thích).

Kết luận
Essential 1 & 2 giúp mình có khung tư duy vững cho JavaScript: hiểu giá trị/tham chiếu, scope/hoisting, this, và mô hình bất đồng bộ. Từ đây, việc học DOM nâng cao, API trình duyệt, hoặc đi vào framework sẽ nhanh hơn rất nhiều.

Nếu bạn đang bắt đầu: cứ đi Essentials → mini-projects → viết blog. Mọi thứ sẽ sáng dần lên theo từng bước nhỏ ✨
